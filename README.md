# 酒狐悄悄话增强版 v2.3 WineFox-Daily

> 一只可爱的酒狐女仆，随时随地为主人送上暖心悄悄话。

基于 [Koishi](https://koishi.chat/) 框架 + OneBot 协议的 QQ 机器人插件。从原版单文件脚本全面升级，现已演进到 **v2.3 经济体系重设计版**，包含时间感知、10级好感、双货币（好感度 + 狐狐券）、签到、酿酒、商店、问答、委托、每日免费指令等丰富功能。

---

## 功能一览

| 功能 | 说明 |
|------|------|
| **时间感知** | 根据当前时段智能推送风格匹配的语录 |
| **好感度系统** | **10个等级**、闲置衰减、升级台词、进度播报、专属称呼 |
| **双货币体系** | 好感度只用于羁绊成长；**狐狐券** 用于商店、酿酒与经济循环 |

| **MC问答** | 4分类100题知识问答，30秒限时，答对给好感与狐狐券 |
| **回忆录** | 初次相遇、升级里程碑、成就时间线 |
| **好感互动** | 摸头/拥抱/告白/喂酒/挠耳朵/牵手，带冷却与骚扰惩罚 |
| **成就系统** | 23项成就徽章，解锁有好感奖励与狐狐券奖励 |
| **稀有语录** | 语录抽卡系统，稀有掉落收录图鉴 |
| **每日一句** | 同一天固定语录 |
| **分类查询/搜索** | 按主题分类浏览或关键词搜索 |
| **节日检测** | 自动识别节日，优先推送节日语录 |
| **占卜/心情** | 每日运势 + 6种心情状态修饰语录 |
| **小游戏** | 猜拳、猜数字(1-100)、御神签 |
| **故事系统** | 100篇/9分类冒险日记，支持分类浏览 |

| **被动触发** | 戳一戳响应 + 群聊关键词冒泡 |
| **每日免费体验** | 每天随机抽取 4 个低等级指令，所有人可无视门槛体验 |
| **酒狐委托** | 每日3个任务，自动追踪进度，完成获得狐狐券 |
| **用户投稿** | 用户投稿 + 管理员审核 |

## 安装

将 `WineFox-Daily` 文件夹放入 Koishi 项目的插件目录中，然后在 Koishi 控制台启用该插件。

```
你的Koishi项目/
├── plugins/
│   └── WineFox-Daily/    <-- 放在这里
│       ├── index.js
│       ├── package.json
│       ├── runtime_config.js
│       ├── lib/
│       ├── data/
│       └── memory/
└── koishi.yml
```

### 经济与成长机制说明

#### 1. 双货币
- **好感度**：只负责角色关系成长，不再作为商店消费货币
- **狐狐券**：通过签到、问答、猜拳、猜数字、开瓶、委托、被动触发、成就等方式获得，用于商店与酿酒消费

#### 2. 10级好感
v2.3 将旧版 6 级好感扩展为 **10 级**，并细化了成长节奏与解锁内容。每次获得或失去好感时，都会显示：
- 当前好感值
- 下一等级名称
- 距离下一等级还差多少
- 下一等级会解锁什么权益

#### 3. 好感下降机制
为了让成长更有张力，v2.3 增加了可预期、可避免的下降机制：
- 长时间不互动会按天数衰减
- 问答答错会扣少量好感
- 酿好的酒放太久会变质并扣好感
- 冷却内反复骚扰互动会先警告，再扣好感

#### 4. 装备机制效果
装备不再只是文案装饰。不同装备会提供不同类型的系统加成，例如：
- 互动额外好感
- 签到额外狐狐券
- 酿酒时间缩减
- 被动冒泡概率提升
- 每日好感上限提升
- 衰减免疫

#### 5. 每日免费体验
每天会从低等级门槛指令中**随机选出 4 个**，允许所有玩家即使好感等级不足也能使用。当群里当天首次触发时，酒狐会自动公告今日免费体验内容。

## 配置

在 `runtime_config.js` 中可配置 23 个参数，涵盖基础、心情、游戏、互动、送礼、收藏、签到等分区。详见文件内的中文注释。

配置加载优先级: `runtime_config.js` > `runtime_config.json`(兼容旧版) > Koishi 后台配置 > Schema 默认值

## 从旧版迁移

如果你之前使用的是单文件版：

1. 将原来的 `酒狐文本存储foxqqh用.txt` 复制到 `WineFox-Daily/data/quotes.txt`
2. 在 Koishi 中禁用旧插件，启用新插件
3. 语录文件格式完全兼容，无需修改
