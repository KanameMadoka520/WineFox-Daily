# 酒狐悄悄话增强版 v2.3 WineFox-Daily

> 一只可爱的酒狐女仆，随时随地为主人送上暖心悄悄话。

基于 [Koishi](https://koishi.chat/) 框架 + OneBot 协议的 QQ 机器人插件。当前主线已经完成 **v2.3 经济体系重设计**，并补上了 **图片卡片输出、主题切换、宽屏帮助/商店布局** 等交互层能力。

| 功能 | 说明 |

| **委托系统** | 每日 3 个任务，自动追踪进度，完成后发放狐狐券 |

稀有语录相关说明：

- 当前主语录库已包含 **28 条稀有语录**
- `酒狐` 指令支持正常稀有掉落
- `酒狐占卜` 与 `酒狐天气` 支持特殊稀有掉落
- 新掉落会自动收录至 `酒狐图鉴`

当前掉落概率：

- `酒狐`：基础 `5%`，再叠加部分装备 bonus
- `酒狐占卜`：幸运值 `>= 85` 时 `18%`，否则 `5%`
- `酒狐天气`：天气为 `极光` / `星空璀璨` / `雨后彩虹` 时 `18%`，否则 `4%`

当前奖励方式：

- 稀有语录本身不会直接发放狐狐券
- 首次解锁会收录到 `酒狐图鉴`
- 同时会推进稀有语录相关成就（`语录收藏家` / `语录猎人`），并通过成就发放好感奖励

说明：

- 插件会优先尝试图片输出
- 若宿主未启用 Puppeteer 服务，或渲染失败，会自动回退为文字
- 后台已补充统一诊断日志，便于排查图片渲染问题

## 安装

将 `WineFox-Daily` 文件夹放入 Koishi 项目的插件目录中，然后在 Koishi 控制台启用该插件。

```
你的Koishi项目/
├── plugins/
│   └── WineFox-Daily/
│       ├── index.js
│       ├── package.json
│       ├── runtime_config.js
│       ├── lib/
│       ├── data/
│       └── memory/
└── koishi.yml
```

### 关于 `koishi`

本插件在代码里只直接依赖 `koishi`，并以 `peerDependencies` 的形式声明。  
通常情况下，**不需要**在本插件目录里单独保留 `node_modules/` 或 `package-lock.json`，使用宿主 Koishi 项目的依赖即可。

### 关于图片功能

图片卡片依赖宿主环境已经安装并启用 Puppeteer 服务插件。  
WineFox 侧已声明 `puppeteer` 为可选注入服务；宿主若未启用 Puppeteer，插件会自动回退为文字输出。

## 经济与成长说明

### 双货币

- **好感度**：只负责角色关系成长和等级解锁
- **狐狐券**：通过签到、问答、猜拳、猜数字、开瓶、委托、被动触发、成就等方式获得，用于商店和酿酒消费

### 日常奖励节奏

当前主线已经按“玩家每天活跃约 10 分钟，也能拿到明显正反馈”做过一轮平衡调整：

- 高频玩法改为 **单次奖励更明显**
- 同时通过每日领取上限，避免单一玩法刷穿经济
- 例如：
  - `每日酒狐`：`+2` 狐狐券（每日 1 次）
  - `酒狐抽签`：`+2` 狐狐券（每日 1 次）
  - `酒狐故事`：单次 `+2` 狐狐券（每日限次）
  - `酒狐猜拳` 胜利：`+3` 狐狐券（每日限次）
  - `酒狐猜数` 胜利：`+4 ~ +6` 狐狐券（每日限次）
  - `酒狐问答` 答对：`+4` 狐狐券（每日限次）
  - `酒狐签到`：基础狐狐券奖励已提升

### 10级好感

v2.3 已将旧版 6 级好感扩展为 **10 级**，并细化成长节奏。  
好感变化后会显示：

- 当前好感值
- 下一等级名称
- 距离下一等级所需点数
- 下一等级解锁内容

如果当天已经达到好感上限，系统会明确提示“今日好感已达到上限”，不会再继续误导性地显示本次已经加了好感。

### 每日免费体验

每天会从低等级门槛指令中随机选出 4 个指令，允许玩家临时无视等级门槛使用。  
帮助菜单和群内首次公告会同步显示当日免费池。

### 装备机制 bonus

装备不再只是文案变化。当前已接入的 bonus 类型包括：

- 互动额外好感
- 签到额外狐狐券
- 酿酒时间缩减
- 酿酒品质提升
- 被动冒泡额外收益
- 被动冒泡概率提升

- 委托额外狐狐券
- 互动冷却缩短
- 每日好感上限提升

### 消耗品效果

当前商店中的消耗品效果已经不再只是文案摆设，主线已经补齐：

- 情绪变化（开心 / 慵懒 / 微醺）

## 从旧版迁移

如果你之前使用的是单文件版：

1. 将原来的 `酒狐文本存储foxqqh用.txt` 复制到 `WineFox-Daily/data/quotes.txt`
2. 在 Koishi 中禁用旧插件，启用新插件
3. 语录文件格式兼容，无需额外修改
